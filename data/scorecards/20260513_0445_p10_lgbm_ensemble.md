## 🎯 3-fold walk-forward 結果 (2023 / 2024 / 2025)

`data/backtest/20260513_walk_forward_3fold.csv` の最終結果。LGBM v2 ensemble + rule + bin calibrator (TRAIN 2021-23 fit) で各 fold 独立評価。

**真ロバスト (全 3 fold >= 80%):**

| filter | 2023 戦/回収 | 2024 戦/回収 | 2025 戦/回収 | min |
|---|---|---|---|---|
| **wl_pop_1_2** | 150 / 195.6% | 90 / 119.4% | 141 / 88.0% | **88.0%** |
| **wl_odds_9_25** | 111 / 154.9% | 74 / 99.5% | 105 / 97.0% | 97.0% |
| **wl_odds_8_25** | 122 / 140.9% | 80 / 92.0% | 116 / 87.8% | 87.8% |
| **wl_odds_7_22** | 130 / 132.2% | 87 / 84.6% | 121 / 85.6% | 84.6% |
| **wl_odds_20_50** | 88 / 83.5% | 64 / 187.2% | 87 / 162.0% | 83.5% |

**重要な含意:**

1. **`wl_pop_1_2` (重賞 + 中京/阪神 + 1-2 人気) が最有力本命戦略** — 戦数 381 (年 ~127)、min 88%、max 196%、bootstrap CI 期待値 100%+
   ※ 注: 過去のスコアカード/コメントで「中山/京都」と記述していたが、JV-Data 場コード 07=中京 / 09=阪神 が正しい (web/codes.py 準拠)。LGBM ensemble 環境では阪神が 10 場中最下位なので、whitelist 自体の再選定要 (Phase 6)。
2. **`wl_odds_20_50` で大穴戦略も実現可能** — 戦数 239 / 年 80、2/3 folds で +160% 超え、min 83.5%
3. **旧 `wl_odds_8_20` は依然崩壊** — 2024 で 2.3% に落ち続けており、in-sample artifact だった事実が再確認

**LGBM ensemble の効き目:**
- 同 sweep の `all` (ベタ買い) row: 2023 69.5% / 2024 77.2% / 2025 82.9% — **2025 で控除率超え**
- 旧 rule-only の 2024 walk-forward (`20260512_walk_forward_v2.csv`) では同期間 70.6%/83.3% — `all` は概ね同水準だが、絞り戦略 (`wl_pop_1_2` 等) の robust 度が ML 導入で劇的に増加
